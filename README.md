# vue-bitbucket-collapsible

## About

This is a simple collapsible pattern inspired by the collapsibles used
in Bitbucket.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
